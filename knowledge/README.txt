# Haruko Wissensdatenbank

Lege hier PDF-Dateien ab, damit Haruko sie lesen kann.

Beispiele:
- Bedienungsanleitungen
- Rezepte
- Verträge
- Persönliche Infos

Haruko scannt diesen Ordner beim Start und nutzt den Inhalt, um Fragen zu beantworten.
Das System nutzt "RAG" (Retrieval Augmented Generation), um relevante Textstellen zu finden.
